# Security Policy

## Supported versions

Only the latest released version of `autohupr` is supported. Fixes are shipped
in a new release rather than backported to older versions.

## Reporting a vulnerability

Please report security vulnerabilities **privately** through GitHub's private
vulnerability reporting:

1. Go to the [Security tab](https://github.com/schubydoo/autohupr/security) of
   this repository.
2. Click **"Report a vulnerability"** and complete the advisory form.

Do not open public issues or pull requests for security reports, and do not
disclose the issue elsewhere until a fix has been released.

You can expect an initial acknowledgement within 7 days. If the report is
confirmed, a fix will be released and the advisory published with credit to the
reporter (unless anonymity is requested).
